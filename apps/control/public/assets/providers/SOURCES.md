# Provider icon sources

All third-party artwork is checked into the repository so the Provider catalog
does not depend on remote images at runtime. Product names and marks remain the
property of their respective owners; inclusion does not imply endorsement.

## Brand assets

- `anthropic.svg`, `gemini.svg`, `deepseek.svg`, `minimax.svg`, `baidu.svg`,
  `volcengine.svg`, `nvidia.svg`, `vertex.svg`, `openrouter.svg`, `ollama.svg`,
  `vllm.svg`, and `huggingface.svg`: downloaded from the
  [Simple Icons CDN](https://cdn.simpleicons.org/) using the corresponding
  Anthropic, Google Gemini, DeepSeek, MiniMax, Baidu, ByteDance, NVIDIA, Google
  Cloud, OpenRouter, Ollama, vLLM, and Hugging Face slugs. Simple Icons data is
  CC0; see its [trademark disclaimer](https://github.com/simple-icons/simple-icons/blob/develop/DISCLAIMER.md).
- `qwen.svg`: downloaded from [Wikimedia Commons](https://commons.wikimedia.org/wiki/File:Qwen_Logo.svg),
  where the asset is attributed to Alibaba Cloud and sourced from the official
  Qwen repository. Its view box is cropped locally to the symbol for small UI use.
- `zai.svg`: downloaded from the [Z.AI Wikimedia Commons asset](https://commons.wikimedia.org/wiki/File:Z.ai_(company_logo).svg).
- `azure.svg`: Microsoft Azure mark downloaded from the
  [theSVG Azure asset](https://thesvg.org/icon/azure).
- `aws.svg`: Amazon Bedrock service icon downloaded from the
  [AWS Architecture icon mirror](https://thesvg.org/icon/aws-amazon-bedrock);
  AWS publishes the source icon set on its
  [Architecture Icons page](https://aws.amazon.com/architecture/icons/).
- `openai.svg`: OpenAI logo geometry from the OpenAI brand asset mirrored by
  Wikimedia Commons. Use is subject to the [OpenAI brand guidelines](https://openai.com/brand/).
- `kimi.svg`: Simple Icons `kimi`, downloaded from the official repository.

## Neutral assets

- `custom.svg` and `custom-anthropic.svg`: TaskLattice-owned neutral interface
  icons for custom compatible endpoints; they are not third-party trademarks.
